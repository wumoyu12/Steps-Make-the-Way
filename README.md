## Overview

This website is a simulated platform designed for a non-profit volunteering organization.

It allows administrators to manage organization information, create volunteering events, and list charitable products.  
Users can create accounts, join the organization, explore available events, and support causes by participating or making purchases.

**Built with:** Python (Flask), HTML, CSS, and JavaScript


## Project Structure

- **code/** – source code for demonstration(HTML, CSS, JavaScript, Flask)
- **images/** – images that are used throughout the website
- **docs/** – supporting documents (flowchart, storyboard, test plan)
- **archive/** – downloadable ZIP version of the project
